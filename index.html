FreshTrack - Sistema de Control para Productos Frescos

HTML CSS JavaScript Arduino Sensores IoT

FreshTrack es un sistema integral de seguimiento y gestión de inventario para productos frescos que incluye sensores de temperatura y humedad conectados a través de Arduino. El sistema proporciona alertas de caducidad en tiempo real y genera reportes detallados sobre el estado de los productos.

Esta solución combina hardware IoT con una interfaz web para monitorear constantemente las condiciones ambientales de almacenamiento, asegurando la calidad y reduciendo el desperdicio de productos perecederos.

Características principales:

  • Monitoreo en tiempo real: Sensores de temperatura y humedad conectados a Arduino
  • Pantallas LED: Muestran información de estado directamente en las zonas de almacenamiento
  • Sistema de alertas: Notificaciones por email y SMS cuando los parámetros salen del rango óptimo
  • Dashboard web: Interfaz intuitiva para visualizar datos históricos y tendencias
  • Gestión de inventario: Seguimiento de fechas de caducidad y rotación de productos

Implementación técnica:

El sistema se basa en una red de sensores DHT22 (temperatura/humedad) conectados a placas Arduino que transmiten datos a un servidor central. La interfaz web está construida con HTML, CSS y JavaScript, y proporciona visualización en tiempo real mediante gráficos interactivos.

Software Project Master - Gestión de Tareas Empresariales

JavaScript React Next.js MySQL JWT TailwindCSS

Software Project Master es un sistema completo que reemplaza la pizarra de trabajo empresarial tradicional, permitiendo a los empleados consultar y actualizar sus tareas diarias de manera digital. Los supervisores pueden asignar, modificar y hacer seguimiento de las tareas de forma eficiente.

El sistema incluye notificaciones automáticas y la generación de reportes detallados sobre el estado de las tareas, mejorando la comunicación, organización y productividad dentro de la empresa.

Arquitectura del proyecto:

  • Frontend: React con Next.js para una experiencia de usuario optimizada
  • Backend: API Routes de Next.js para todas las operaciones CRUD
  • Autenticación: Sistema basado en JWT con roles (admin y empleado)
  • Base de datos: MySQL para almacenar usuarios y tareas
  • UI/UX: Diseño moderno con TailwindCSS e iconos de Lucide React

Características principales:

  • Sistema Kanban: Gestión visual de tareas con funcionalidad drag-and-drop
  • Estados de tarea: Pendiente, En Progreso, En Revisión, Completada
  • Prioridades: Alta, Media, Baja con indicadores visuales
  • Reportes: Generación de informes PDF globales y por usuario
  • Dashboard: Vista personalizada para cada empleado y vista general para supervisores

Sistema de Inventario UNPHU - Gestión de Préstamos Institucionales

Next.js React NextAuth.js MySQL TailwindCSS

Este proyecto desarrolla un sistema informático para la gestión eficiente de préstamos y devoluciones de materiales tecnológicos a estudiantes y docentes en la Universidad Nacional Pedro Henríquez Ureña (UNPHU).

La solución automatiza y optimiza los procesos de los departamentos de tecnología y audiovisual, mejorando el método de gestión de préstamos de materiales de uso tecnológico e inventario, adaptable a cualquier organización con necesidades similares.

Características del Sistema:

  • Gestión de Inventario: Registro y seguimiento de equipos por departamento
  • Sistema de Solicitudes: Proceso de solicitud de préstamos con flujo de aprobación
  • Control de Estados: Seguimiento de disponibilidad (disponible, agotado, en mantenimiento)
  • Panel Administrativo: Gestión centralizada con estadísticas de uso
  • Experiencia de Usuario: Interfaces intuitivas con retroalimentación en tiempo real

Arquitectura técnica:

El sistema sigue una arquitectura moderna utilizando Next.js como framework full-stack, con rutas API para el backend y componentes React para la interfaz de usuario. La autenticación se gestiona a través de NextAuth.js con un sistema de roles, y Tailwind CSS proporciona un diseño responsivo y atractivo.

Base de Datos:

Utiliza MySQL con relaciones entre tablas para usuarios, departamentos, items y solicitudes, permitiendo consultas eficientes y manteniendo la integridad referencial de los datos.
